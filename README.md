# HYPERLINKFOUNDATION
## A place for links to live.


Using:
=============

1. clone the repo
2. install node [npm install]
3. run app [app.js]


In Terminal run these two lines to install node and run the app.

>npm install

>node app.js

We are using [Chat Websocket](https://github.com/feiyunruyue/chat-websocket) for our chat.
